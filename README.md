# Stock Ticker Demo

A simple **Stock Price Ticker** web app built with **Python (Flask)** and **HTML/CSS/JavaScript**.  
It simulates live stock prices with random values, arrows, and color indicators updated every 2 seconds.

## Features

- Simulated stock prices for Apple, Google, Microsoft, and Berkshire  
- Arrows indicate price increase (`↑`) or decrease (`↓`)  
- Color-coded prices: green = up, red = down, black = unchanged  
- Timestamp showing the last update  

### Prerequisites

1. Make sure **Python 3** is installed. 
2. Make sure Git is installed

### Installation & Running

1. Clone the repo:
```bash
git clone https://github.com/mahitha-muppalaneni/stock-price-ticker-flask.git
cd stock-price-ticker-flask

2. Install Flask
```bash
pip install flask

3. Run the app:
python app.py

4. Open a browser at:
http://127.0.0.1:5000
